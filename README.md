# MinimizeMute

## English

MinimizeMute is a small Windows utility that automatically mutes selected apps when they are minimized, then restores their previous volume when they are shown again.

### Download

Download `MinimizeMuteSetup.exe` from Releases:

https://github.com/Dcvu233/MinimizeMute/releases

Double-click the installer to install it.

### Features

- Mute selected apps when minimized
- Restore the previous volume/mute state when restored
- Show currently running window apps
- Support multi-process apps such as Chrome
- Optionally mute all apps when minimized
- Minimize to the system tray instead of exiting

### How To Use

1. Open MinimizeMute.
2. Select an app from the app list on the left.
3. Click "加入最小化静音".
4. The selected app will be muted when minimized and restored when shown again.

If "所有程序最小化时静音" is enabled, MinimizeMute will apply the rule to all running window apps.

### Tray

Clicking the close button hides MinimizeMute to the system tray instead of exiting.

- Double-click the tray icon: show the window
- Right-click the tray icon: show the window or exit

### Requirements

- Windows 10 or later
- 64-bit Windows

## 中文

MinimizeMute 是一个 Windows 小工具：把指定应用加入列表后，当这个应用最小化时自动静音，恢复窗口后自动还原原来的音量。

### 下载

在 Releases 页面下载 `MinimizeMuteSetup.exe`：

https://github.com/Dcvu233/MinimizeMute/releases

下载后双击安装即可。

### 主要功能

- 选择指定应用，在最小化时自动静音
- 恢复窗口后还原原来的音量/静音状态
- 显示当前正在运行的应用列表
- 支持 Chrome 等多进程应用
- 可选择让所有程序在最小化时静音
- 点击关闭按钮时最小化到系统托盘

### 使用方法

1. 打开 MinimizeMute。
2. 在左侧应用列表里选中一个应用。
3. 点击“加入最小化静音”。
4. 之后这个应用最小化时会自动静音，恢复后会自动还原音量。

如果勾选“所有程序最小化时静音”，则会对所有正在运行的窗口程序生效。

### 托盘

点击窗口右上角关闭按钮时，程序不会退出，而是隐藏到系统托盘。

- 双击托盘图标：显示窗口
- 右键托盘图标：显示窗口或退出程序

### 系统要求

- Windows 10 或更新版本
- 64 位系统
